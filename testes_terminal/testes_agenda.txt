1. TESTAR O CHAT PRINCIPAL COM O GEMMA
============================================================

Essa função testa o chat principal do projeto.

- iniciar o JARVIS;
- receber uma mensagem digitada pelo usuário;
- chamar a LLM;
- mostrar a resposta no terminal.

============================================================

Ela verifica se o Python consegue abrir o arquivo da agenda e transformar o conteúdo JSON em uma lista de compromissos.

Comando simples:

python -c "from src.agenda import carregar_agenda; print(carregar_agenda())"

============================================================
3. TESTAR A FUNÇÃO recuperar_compromissos_por_data()
============================================================

Essa função recupera compromissos de uma data específica, mas ainda retorna no formato bruto de lista.

Ela é uma função intermediária.

Isso significa que ela apenas pega os dados da agenda, filtra pela data escolhida e devolve uma lista de compromissos encontrados.

Comando para uma data com compromisso:

python -c "from src.agenda import recuperar_compromissos_por_data; print(recuperar_compromissos_por_data('2026-05-15'))"

Comando para uma data sem compromisso:

python -c "from src.agenda import recuperar_compromissos_por_data; print(recuperar_compromissos_por_data('2026-05-16'))"


============================================================
4. TESTAR A FUNÇÃO formatar_compromissos_por_data()
============================================================

Essa função transforma a lista de compromissos de uma data específica em um texto mais bonito.

Ela deve ser usada depois da função recuperar_compromissos_por_data(), que recupera os compromissos daquela data.

Esse teste verifica se o sistema consegue pegar os dados brutos da agenda e transformar em uma resposta legível para o usuário.

Comando:

python -c "from src.agenda import recuperar_compromissos_por_data, formatar_compromissos_por_data; compromissos = recuperar_compromissos_por_data('2026-05-15'); print(formatar_compromissos_por_data(compromissos, '2026-05-15'))"


============================================================
5. TESTAR A FUNÇÃO consultar_agenda_por_data()
============================================================

Essa função já recupera os compromissos de uma data específica e devolve a resposta formatada.

Ela junta duas etapas:

1. recuperar_compromissos_por_data()
   recupera os compromissos daquela data.

2. formatar_compromissos_por_data()
   transforma a lista encontrada em uma resposta bonita.

Comando para uma data com compromisso:

python -c "from src.agenda import consultar_agenda_por_data; print(consultar_agenda_por_data('2026-05-15'))"

Comando para uma data sem compromisso:

python -c "from src.agenda import consultar_agenda_por_data; print(consultar_agenda_por_data('2026-05-16'))"
